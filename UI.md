# Cписок интерфейсов

**Гость**
 - Регистрация аккаунта
 - Вход в аккаунт
 - Восстановление пароля
 - Сброс пароля

**Гость/пользователь (доступные обоим интерфейсы):**

 - Главная страница
 - Калькулятор доходов
 - Топ блоггеров
 - Страница блоггера в топе

**Пользователь:**

 - Список доступных отчётов
 - Просмотр отчета
 - Заказ отчёта
 - Окно платежной системы (для оплаты отчета)
